# Phase 38 Console Sections + Messaging Cleanup

The Driver Console is now separated into two screens behind a hamburger menu.

## Console screens

```txt
Communication
- Passenger Requests
- Driver Translation status
- Driver Messages
- Preset translated popups
- Typed custom passenger message

Settings
- Notification Sounds
- System status
```

## Driver Messages

Preset translated messages now live with the driver message popup controls.

The driver can also type a custom message to the passenger.

## Custom message limitation

Typed custom messages are sent as typed and are not machine-translated yet.

Preset messages remain manually translated and use the passenger-selected language when possible.
