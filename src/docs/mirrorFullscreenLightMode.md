# Mirror Full-Screen Light Mode

When Mirror is active, the page now switches to a full-screen white light surface:

```txt
White background fills the full viewport
Camera preview remains visible in the middle
Stop Mirror button floats at the bottom
Normal Mirror card returns after stopping
```

This creates more usable light than only turning the card interior white.

## Browser limitation

The app still cannot directly set device system brightness in a standard browser. The full-screen white surface is the browser-safe way to make the screen itself act as a light source.
